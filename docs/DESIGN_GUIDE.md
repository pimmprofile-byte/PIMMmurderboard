# 디자인 · 연출 · 진행방식 가이드 — 《동창회 : 폐교》

빌드 기준선. Cowork 콘텐츠가 들어오기 전, **화면·무드·진행 뼈대**를 여기서 고정한다.

---

## 1. 무드 & 아트디렉션 (이미지 없음, 타이포·색·연출로)

- **레퍼런스:** PIMM '혼문(魂門)' 시네마틱(다크+명조체+비네팅) × 카카오/디스코드 오픈채팅 말풍선.
- **콘셉트:** *비 내리는 폐교의 새벽, 기억이 흐릿한 망령들.* 서늘함 + 먼지 낀 형광등 + 낡은 칠판.
- **팔레트**
  - 배경: 근검정 `#0b0a0d` ~ `#111015`, 폐교의 청록빛 냉기 `#6f8ea3`.
  - 포인트: 핏빛 앰버 `#c8442f`(오승택/사건), 낡은 금 `#c9a24a`(기억·온기), 형광 청록 `#79c6c0`(망령).
  - 인물별 고유색(배지·말풍선 네임).
- **타이포:** 타이틀/공통지문 = 명조(Nanum Myeongjo). 본문·채팅 = 시스템 산세리프(가독성 우선).
- **모션(절제):** 타이틀 비네팅·엠버 글로우, 라운드 전환 페이드, "기억의 균열" 시 미세한 글리치/흔들림.
  과하지 않게 — 텍스트 가독이 최우선.

## 2. 화면 흐름

```
① 타이틀        시네마틱. 제목 + 공통지문 도입 + '들어가기'
② 로비          배역 선택(각 기기). 빈 자리 'AI로 채우기'. 배역 확정 시 내 시트 준비
③ 시트(모달)    내 배역 대본(비공개·자기 기기에만). 죄/목표/기억의 균열
④ 테이블        오픈채팅. 공통 사건 로그 + 인물 말풍선. 'AI 배역 부르기'
                라운드 진행 바 / 조사카드 열람 / 오승택 망령 개입 내레이션
⑤ 자백·지목     "나는 누구에게 죄가 있다 보는가 / 나 자신은?" — 자기 지목(자백) 중심
⑥ 엔딩          진혼(전원 자백) / 미완 / 루프 배드엔딩 등. 진상 서술 공개
```

## 3. 연출 장치 (이 시나리오 전용)

- **공통지문 낭독:** 타이틀→테이블 진입 시 도입 내레이션을 천천히 노출(타이핑/페이드).
- **기억의 균열:** 라운드가 진행될수록 각자 시트의 "기억의 균열" 섹션이 **한 조각씩 선명해짐**
  (처음엔 흐릿 → 죄가 드러남). 자백 트리거와 연동.
- **오승택 망령의 개입:** 라운드 사이, 전체 공개 내레이션으로 짧게 등장(누구인지 모른 채 서늘함).
  후반에 정체·의도가 드러남.
- **타살 미끼:** 전반부 조사카드는 "누가 죽였나"를 의심하게 하는 정황 위주 → 후반 반전 카드로 전복.
- **자백의 순간:** 인물이 자기 죄를 인정하면 말풍선/화면에 정적인 강조(핏빛→금빛 전환) 연출.

## 4. 진행방식 (80분 기준, GM 없이 반자동)

- **라운드제(4~5R 권장, Cowork가 확정).** 각 라운드: 도입 내레이션 → 조사·대화 → (필요시 조사카드 공개)
  → 다음 라운드. 진행 바 + '다음 라운드' 버튼(누구나 진행 가능한 협동식).
- **대화:** 오픈채팅. 사람은 자기 배역으로 발언, AI 배역은 '부르기'로 끼어듦(수동 트리거로 페이스 조절).
- **자백·지목(엔딩 트리거):** 기존 '범인 지목'을 **역구조**로 교체 —
  - 각 인물은 `타인의 죄 지목` + `자기 죄 인정(자백)`을 선택.
  - **전원이 자기 자신을 지목(자백)** → 진혼(최고 엔딩). 부분 인정/서로 남 탓 → 하위 엔딩.
  - 자백을 **어렵게**: 관련 조사카드·기억 조각이 충분히 열려야 자백 가능(옵션, [열린 질문 3]).
- **점수판:** 은폐/드러냄/자백을 합산해 엔딩과 함께 표시(정확한 규칙은 Cowork 8번).

## 5. 엔진 변경 메모 (콘텐츠 확정 후 반영)

- `scenario.py`: `is_culprit/CULPRIT_ID` 단일 범인 모델 → **인물별 `sin`(죄)·`confessTrigger`·`goals`**
  구조로 확장. `SOLUTION` → 다중 엔딩(`endings[]` + 조건).
- `server.py`: `/api/vote`(범인 1인) → `/api/accuse`(타인 지목) + `/api/confess`(자기 인정),
  엔딩 판정 로직(전원 자백=진혼). 라운드별 조사카드 공개 상태를 방 상태에 포함.
- `index.html`: 투표 모달 → **자백/지목 화면**, '기억의 균열' 점진 노출, 망령 내레이션 배너.

> 위 변경은 현재 검증된 방(room)·폴링·LAN·AI연기 인프라 위에 얹으면 되므로 리스크 낮음.
