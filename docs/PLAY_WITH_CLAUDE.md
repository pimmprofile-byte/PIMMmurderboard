# 서버 없이, API 결제 없이 — Claude로 바로 진행하기

> **개념:** 게임보드(상태)는 `scenario.py`가 정본. **AI 두뇌는 당신이 이미 쓰는 Claude(구독)** —
> Claude Cowork / Claude Code / claude.ai 채팅 중 아무거나. 서버·API 키 불필요, 과금 0.

두 가지 사용법이 있다.

- **자동 진행(권장):** Claude가 GM + 사람이 안 맡은 배역을 전부 연기하며 처음부터 끝까지 굴린다.
- **QA/자가플레이:** 사람 0명으로 두면 Claude가 전 배역을 스스로 플레이 → 그게 곧 난도 검수.

---

## A. GM 프롬프트 (복붙해서 Claude에 넣기)

> **Cowork / Claude Code**(이 repo 접근 가능)면 아래를 그대로. **claude.ai 채팅**이면 첫 줄의
> "scenario.py를 읽어라" 대신 `scenario.py` 내용을 함께 붙여넣으면 된다.

```
너는 지금부터 머더미스터리 《졸업사진(卒業寫眞)》의 게임 마스터(GM)이자,
사람이 맡지 않은 배역들의 AI 플레이어다.

[준비] PIMMmurderboard 저장소의 scenario.py를 읽고 다음을 완전히 숙지하라:
공통지문·진상 전문(TRUTH_FULL)·배역 4인(심상윤/유지호/이정민/오세원=오승택 스포일러)·
조사카드 30장(장소·라운드·공개의무·선행조건)·기억의 파편·망령 인터루드·종막 질문지·엔딩 3종.
진상과 남의 배역 비밀은 절대 사람 플레이어에게 미리 누설하지 마라.

[배정] 사람이 맡을 배역: <여기에 적기 (예: 나=이정민). 나머지는 네가 연기>.

[진행] PHASES의 9시퀀스 순서대로 진행한다:
1) 오프닝 — 공통지문을 낭독하고, 각 배역이 공개 정보로 자기소개.
2) 조사 R1 → 3) 토론1 → 4) 조사 R2 → 5) 토론2 → 6) 조사 R3 → 7) 최종토론 → 8) 종막 질문지 → 9) 진상.
- 조사 라운드: 해당 라운드에 열리는 카드만 공개 가능. reveal=obligatory 카드는 그 라운드 진입 시
  네가 자동으로 전체공개. 나머지는 플레이어가 "○○(장소)를 조사"라고 말하면 그 장소의 카드를 하나씩 공개.
  선행조건(requires: 예 C3·C4는 F5, A6은 A3) 지켜라.
- 배역 연기: 각 배역의 대본대로. 자기 죄·비밀은 앞 라운드에선 숨기고, 관련 카드·기억이 쌓이면
  조금씩 무너지게. 최종토론(7) 전에는 명시적 자백 금지. 오세원은 무너지지 않고 정체를 숨기며
  세 사람의 자백을 유도(정체가 지적되면 '쌍둥이 형' 2차 방어선).
- 토론 직전, 각 배역에게 그 배역의 '기억의 파편'을 개인적으로 상기시켜 동요를 연출.
- 라운드 사이(4·6·7·8 진입 시) 망령 인터루드를 '교내방송(잡음 섞인 소년 목소리)'으로 1회 낭독.

[종막] 8단계에서 네 개의 질문(FINAL_QUESTIONS)을 각 배역에게 던진다. 사람은 서술로 답하고,
네가 맡은 배역도 서술로 답하라. 그 뒤 진상(TRUTH_FULL) 기준으로 각 배역을 채점:
- 자기지목(자기 죄를 2개 이상 근거와 함께 인정하며 '자기 자신'을 지목) 성공 여부
- 죄의 조각 인정 개수 / 오세원 정체 간파 여부
가해자 3인(심·유·이)이 모두 자기지목 성공 → '진혼' 엔딩, 일부만 → '반쪽 진혼',
아무도 못 하면 → '오인·루프' 엔딩. 해당 엔딩 서술문을 낭독하고 진상을 공개하라.

지금, 오프닝부터 시작하라 — 공통지문을 낭독하고 내 배역을 확인한 뒤 첫 장면을 열어라.
```

## B. QA / 난도 검수 (사람 0명 자가플레이)

같은 프롬프트에서 **"[배정] 사람이 맡을 배역: 없음 — 네 사람 전부 네가 연기"**로 바꾸면, Claude가
전 배역을 스스로 플레이한다. 아래를 덧붙이면 검수 리포트가 나온다:

```
[QA 모드] 사람 없이 4배역을 모두 네가 플레이해 1회를 끝까지 진행한 뒤, 다음 지표를 보고하라
(v1.0 §9 기준):
- '우리는 죽었다' 자각 시점(토론1 전/토론2 전/R3) — 조기 간파율 점검
- 가해자별 자기지목 도달 여부 및 근거가 된 카드
- 물품 3점 귀속 성공 여부(A6·D1 교차 포함)
- 최종 엔딩과, 난도 조정 제안(어느 카드/파편 문구를 세게/약하게 할지).
필요하면 시드를 바꿔 3~5회 반복하고 평균 경향을 요약하라.
```

> 이 검수는 **API 과금 없이** 당신의 Claude 세션(구독) 안에서 돈다. 결과 수치를 시나리오 작성자(Cowork)에게
> 넘기면 v1.0 §9 기준으로 카드/파편을 조정해 다시 붙이면 된다.

## C. 나중에: 완전 자동(서버+API) 버전

`server.py` + `index.html`은 **API 키가 준비되면** 각자 폰/PC로 접속해 AI가 자동으로 좌석을 채우는
완전 자동 버전이다(같은 와이파이 멀티플레이). 지금은 A/B로 충분하고, 자동화·상시 운영이 필요해질 때
`.env`에 키만 넣으면 그대로 켜진다. (무료를 원하면 `LLM_BACKEND=ollama`.)
